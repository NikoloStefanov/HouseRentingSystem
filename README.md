# HouseRentingSystem
A project for managing house rentals. The system allows users to browse available houses for rent, add new listings, and make inquiries.

## How to Run the Project

### 1. Clone the Repository.

First, clone the repository to your local machine. Run the following command in your terminal:

```bash
git clone https://github.com/NikoloStefanov/HouseRentingSystem.git
cd HouseRentingSystem

### 2. Install Dependencies

dotnet restore

### 3. Run the Project

dotnet run

### 4.Database and migrations

dotnet ef database update
